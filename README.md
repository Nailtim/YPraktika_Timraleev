# Название

Система управления камерами хранения (Locker Management System)

<img width="908" height="887" alt="image" src="https://github.com/user-attachments/assets/90661a96-2c4c-4006-b7ae-d9860b9ca80a" />


## Описание
«Система управления камерами хранения» — это десктопное приложение для автоматизации работы автоматизированной камеры хранения, разработанное в рамках производственной практики на предприятии. Приложение позволяет пользователям авторизоваться, арендовать свободную ячейку, указав PIN-код и описание вещей, освобождать ячейку с проверкой PIN-кода, а также получать фискальный чек с деталями операции. Система поддерживает многопользовательский режим и разграничение доступа через логин и пароль.

## Установка
Убедитесь, что на компьютере установлена IntelliJ IDEA Community Edition 2024.2.2 (или другая среда разработки Java).

Скачайте файлы приложения:

- `LockerSystem.java`
- `sqlite-jdbc.jar` (драйвер базы данных)

Добавьте `sqlite-jdbc.jar` в зависимости вашего проекта:

**File → Project Structure → Libraries → + → Java**

Выберите файл `sqlite-jdbc.jar`.

Скомпилируйте приложение `LockerSystem.java`.

Запустите приложение.

При первом запуске автоматически создастся файл базы данных `lockers.db`, а также:

- будут созданы таблицы `users` и `lockers`;
- добавлены 16 тестовых ячеек (с ID от 1 до 16) со статусом `FREE`;
- создана учётная запись администратора по умолчанию (логин и пароль задаются через интерфейс регистрации).

## Использование
«Система управления камерами хранения» — это надёжное и простое в использовании приложение для автоматизации работы камеры хранения, построенное на локальной базе данных SQLite. Оно предоставляет удобный графический интерфейс для:

- авторизации и регистрации пользователей;
- визуального отображения статуса всех ячеек (занята/свободна) с цветовой индикацией;
- аренды свободной ячейки с указанием PIN-кода и описания вещей;
- освобождения занятой ячейки с обязательным подтверждением PIN-кода;
- автоматической генерации фискального чека с номером ячейки, описанием вещей, датой и временем операции;
- защиты от несанкционированного доступа к вещам через проверку PIN-кода.

Приложение поддерживает многопользовательский режим, что позволяет использовать его в местах с высоким трафиком (вокзалы, аэропорты, ТЦ) с разграничением ответственности между операторами.
